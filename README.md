# Armamentul Militar de Nouă Generație - Teroarea din Inima Soldaților

## Descriere

Mașină controlată prin Bluetooth, cu două motoare DC controlate prin L298N și lansator de mingi de ping-pong orientabil cu servomotor.

## Funcționalități

- Control Bluetooth prin HC-05
- Deplasare înainte/înapoi/stânga/dreapta
- Control viteză motoare prin PWM
- Offset software pentru calibrarea motoarelor
- Orientare lansator cu servomotor SG90
- Lansare minge cu elastic și servomotor
- Temporizare lansator cu Timer2 și întreruperi

## Structura

- src/ - codul sursă
- hardware/ - schema electrică și schema bloc
- images/ - poze cu proiectul
- platformio.ini - configurația PlatformIO

## Componente

- ATmega328P
- HC-05
- L298N
- 2 motoare DC
- 2 servomotoare SG90
- Adaptor 12V / 0.8A
- Breadboard
- Rezistențe 1kΩ
- Condensator 1000 µF
- Sârmă, mingi de ping-pong
